用户可能修改的参数:
1, 广播包内容数据 adv_data
2, 广播包内容长度 LEN_DATA
3, 发射次数 txcnt
4, 接收次数 rxcnt

5, 射频发射功率BLE_TX_POWER
6, 接收超时时间BLE_RX_TIMEOUT,这个时间会影响接收效率和功耗
7, 功耗方面,根据mcu实际情况设置合适的BLE_START_TIME
